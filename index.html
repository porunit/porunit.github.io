Опыт работы

В качестве Java-разработчика я был ответственен за разработку серверной части интернет-магазина, включая следующие задачи:

  • Разработка эндпоинтов для управления товарными позициями, обработки заказов и управления пользовательскими данными. – Интеграция API с клиентской частью сайта.

  • Использование Spring Data JPA для создания и управления схемой базы данных в PostgreSQL. – Разработка запросов для извлечения, добавления, обновления и удаления данных.

  • Имплементация кэширования наиболее часто запрашиваемых данных для оптимизации производительности.

  • Настройка и оптимизация контейнеров для гарантированной и надежной работы в различных средах.

  • Разработка юнит и интеграционных тестов с использованием JUnit и Spring Boot Test.

Проекты

Консольные Шахматы на Java

Проект представляет себя реализацию классической игры в шахматы с консольным интерфейсом. Система базируется на клиент-серверной архитектуре, где сервер обеспечивает взаимодействие двух клиентов, участвующих в игре. Данные между клиентами и сервером передаются по протоколу TCP/IP.


Особенности:


  • Консольный интерфейс
  • Клиент-серверная архитектура
  • Поддержка игры между двумя участниками
  • Использование стандартных средств Java без сторонних библиотек

ConsoleChess

Web API для взаимодействия с Базой Данных

Данный проект - прикладной интерфейс для взаимодействия с базой данных, в виде веб сайта. Сервис дает возможность видеть группы других пользователей, а так же добавлять, удалять и изменять свои группы а также искать наиболее популярные среди них. Кроме того, на сервисе можно смотреть на визуализацию групп на координатной оси, которые расположены там относительно указанных координат.

Интернет магазин Backend

Основные технологии и инструменты:


  • Spring Boot - основа приложения.
  • Data JPA - для работы с базой данных.
  • PostgreSQL - в качестве основной базы данных.
  • Redis - для кеширования данных.
  • Docker - для создания и управления контейнерами.
  • JWT - для реализации системы авторизации.
Проект представялет собой серверную часть полноценного интернет магазина. Поддерживает авторизацию, динамическую систему фильтрации и так далее.
